## 프로젝트 소개

별자리에 대한 정보, 운세를 보여주고 커뮤니티를 이용하는 반응형 웹 사이트

## 기능

`회원가입` : 사용자의 이메일, 비밀번호, 닉네임, 생일을 입력받아 회원가입<br>
`로그인` : 사용자의 이메일과 비밀번호를 입력해서 로그인<br>
`운세 조회` : 로그인한 유저의 생일에 기반한 운세를 출력<br>
`별자리 조회` : 황도 12궁의 이미지와 설명을 출력<br>
`별자리 게시판` : 별자리에 대한 유저들의 게시글들을 간략하게 제공<br>
`게시글 자세히보기` : 원하는 게시글의 상세 내용을 조회<br>
`게시글 작성` : 별자리에 대한 의견과 내용을 작성<br>
`게시글 수정` : 게시글의 원하는 부분만 수정<br>
`게시글 삭제` : 게시글을 삭제<br>
`게시판 제목 검색` : 키워드를 입력하여 키워드로 제목을 검색 가능<br>
`게시판 카테고리 필터` : 게시판 글 중에서 원하는 별자리에 해당하는 게시글만 필터링<br>
`팀 소개` : 프로젝트를 만든 개발자들의 아바타와 설명 조회 가능<br>
